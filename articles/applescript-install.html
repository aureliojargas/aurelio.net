--- layout: 2013-articles #worked: 0 date: 2007-05-22 title: How to install AppleScripts on your Mac tags: [AppleScript, reference] ---

1. Launch AppleScript Utility

2. Enable the menu bar icon

3. Open the User Scripts folder

4. Create the Applications folder on it

5. Create the iTunes, iPhoto, ... folders

6. Move the AppleScripts to the App folder

7. Have Fun!

Appendix A - Command line version

{% highlight shell %} open -a "AppleScript Utility" mkdir -p ~/Library/Scripts/Applications cd $_ mkdir "Address Book" iPhoto iTunes mv ~/Desktop/iTunes*.scpt iTunes # \o/ {% endhighlight %}
