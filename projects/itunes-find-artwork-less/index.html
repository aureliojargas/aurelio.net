You use that nice iTunes script/plugin to automatically download the album cover from the Internet, don’t you? But after using it for a while, how could you know which songs are still missing the artwork?

This script fills this gap, showing you the “artwork-less” songs. Then you can run the download plugin on them.

Road Map

Step 1

Select the desired songs or playlist in iTunes.

Step 2

Run this script.

Step 3

Wait a moment and a new playlist named "* Missing Artwork" will appear. All the songs on it have no Album Artwork.
