Saiu a versão nova 2.5 do txt2tags! Mas esta não é uma versão qualquer, ela é uma versão especial de aniversário. Para comemorar esta incrível marca de sete anos de vida, tão difícil de se atingir com um software livre feito nas horas vagas, quem ganha o presente é você.

Não é um, nem dois, mas são três formatos novos prontinhos para serem usados:

  • Wikipedia
  • Google Code Wiki
  • DokuWiki

Se você vai escrever um artigo para a Wikipedia, atualizar a documentação online do seu programa hospedado no Google Code ou editar uma página no formato DokuWiki, o txt2tags é seu amigo.

Continue usando as **marcas** //que// __você__ ``já`` conhece e fique tranqüilo que o txt2tags se encarregará de convertê-las para estes formatos novos. Os códigos para a opção -t são wiki, gwiki e doku. Divirta-se!

Por falar em marcas, já teve a necessidade de riscar uma palavra? Agora você também pode fazer isso no txt2tags, com a novíssima marca dos hífens, --assim--. Simples, não?

Ainda há outras melhorias como a nova interface PHP, suporte mais robusto ao UTF-8, links locais no LaTeX, ferramentas novas e man page traduzida para chinês (Eu sabia que você ia gostar dessa!)

Outra novidade é um guia que ensina como adicionar um formato novo ao txt2tags. Que tal colaborar com o projeto, fazendo o suporte a algum tipo novo de documento? Não é difícil!

Por falar em colaboração, esta versão nova foi patrocinada por uma generosa doação do usuário dinamarquês Dmitri Popov, que mantém um site de manuais online que já ajudou dezenas de projetos livres. Muito obrigado Dmitri!

Se você já é usuário do txt2tags, atualize-se.

Se você ainda não o conhece, experimente agora mesmo, pela Internet.
