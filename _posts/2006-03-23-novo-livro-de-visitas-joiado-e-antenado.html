O script de visitas ganhou uma garibada geral, agora gerando uma página de visual mais atraente, que facilita a leitura dos recados. Confira o resultado e deixe seu recado! Se você não é nerd, pare a leitura aqui.

Ok, vamos às nerdices: agora o script é modular, gerando apenas o formulário e os recados, podendo ser embutido em qualquer página PHP com um simples require(). O código HTML gerado é limpo e possui DIVs, usando CSS para a formatação. Foi adicionada checagem básica contra Spam, código atualizado para PHP4, e-mail do visitante escondido como tooltip em seu nome, removidos o suporte a paginação e o envio de e-mails de notificação.
