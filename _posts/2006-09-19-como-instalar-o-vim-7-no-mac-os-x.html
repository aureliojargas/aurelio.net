Recentemente foi lançada a versão nova (sete) do melhor editor de textos já escrito: Vim. Algumas das novidades são: verificação ortográfica enquanto digita, autocomplemento de palavras-chave de programação, destaque de pares de colchetes, parênteses e chaves, e abas (tabs). Saiba mais sobre as novidades.

Para brincar com o Vim novo no Mac é fácil, acompanhe:

  1. Vá no site do Rudix e escolha o pacote do Vim. Para facilitar, aqui está o link direto.
  2. Espere baixar o pacote, então dê duplo clique no arquivo vim-7.0-0.dmg.
  3. Dentro desta imagem está o arquivo vim.pkg. Duplo clique nele!
  4. O instalador vai aparecer. Vá apertando o botão “Continuar” até o fim.
  5. Pronto! Vim instalado. Chame-o do Terminal como vim7 e aproveite!

Nota: O Vim 7 é instalado em /usr/local/, não conflitando com o Vim 6.2 (vi) que vem com o Tiger. Os dois funcionarão de maneira independente. Essa é uma das facilidades que o Rudix proporciona: instalação fácil e limpa.

Eu costumo escrever todo o texto no Vim e quando ele está “pronto”, abro o arquivo no Editor de Texto (TextEdit.app) somente para fazer a verificação ortográfica. Agora com o Vim 7 este passo não é mais necessário.

Para ativar a verificação ortográfica nele é um pouco chato, é preciso baixar o arquivo com as palavras, processá-lo, colocar no ~/.vim/spell e configurar o Vim para lê-lo. Ufa! O amigo TaQ descreveu todos estes passos em seu Blog, confira: Usando correção ortográfica no Vim7.

Tudo pronto? Agora enquanto você digita, as palavras erradas ficam com fundo vermelho. De cara ele mostra palavras de uma letra como erradas: a, e, é, à. Assim você já pode treinar como adiciona palavras no dicionário. Cursor sobre ela e aperte zg (g de Good). Caso queira retirar uma palavra do dicionário, faça zw (w de Wrong). Fácil e rápido.

Use os comandos :set spell e :set nospell para ligar e desligar a verificação. Para corrigir uma palavra errada digite z= para ver a lista de possibilidades e digite o número da palavra certa.

O Vim está bloat? Sim, faz anos já… Mas continua veloz como o pensamento, simples como surfar*, e mais poderoso do que nunca!

Surfar é simples. Depois que você aprende, é claro. Primeiro é preciso dominar o básico sobre seu corpo (nadar, remar, perder o medo das ondas, equilibrar-se deitado, sentado e em pé), sobre as ondas (o tempo da onda, saber onde ela vai quebrar, se vai abrir ou não, se é gorda ou cavada, saber furá-la), e sobre sua segurança (saber quem tem a preferência na onda, cuidar com o fundo, as pedras e as quilhas). Não é um aprendizado fácil nem rápido, mas depois que deslancha, é intuitivo :)
