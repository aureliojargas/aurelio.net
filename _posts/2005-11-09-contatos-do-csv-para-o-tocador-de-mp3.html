No final da notícia anterior eu citei que seria fácil adaptar o script Python para ler os contatos de um arquivo CSV. Está feito. Agora usuários de Linux e Windows também podem copiar seus contatos para o tocador de MP3.

Basta ir no seu programa favorito (MS Outlook, Kontact, Yahoo!, …) e exportar os contatos para o formato CSV. Então rode este script e ele gerará o arquivo binário que o tocador entende. Copie este arquivo para o tocador e use a sua função “Import” para carregar os contatos.

Exemplo:
python s1mp3-import-contacts.py -v 2.0 -o outlook contacts.csv

Há três formatos diferentes suportados (-v), tente todos que um deve funcionar no seu tocador. Se o seu CSV precisar de um offset (-o) diferente, avise-me para que eu possa inclui-lo em versões futuras.


Relembrando: isso não é para iPods e outros tocadores de grife. Só funciona com tocadores xingue-lingues que vêm com um programa para Windows chamado TELBOOK.EXE.
