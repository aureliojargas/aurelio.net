Todas as regras de endereço para linhas funcionam igualmente para colunas, bastando colocá-las entre colchetes.

rac "[1:5]" /etc/passwd

Mostra as cinco primeiras colunas de todas as linhas.

A diferença é que para informar colunas soltas, usa-se a vírgula:

rac "[1,3,5,7,9]" /etc/passwd

Este comando mostra as colunas ímpares, até a nona.
