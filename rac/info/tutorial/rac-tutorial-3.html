Usar o programa é muito fácil, basta chamá-lo assim:

rac  ENDEREÇO  ARQUIVO

Como já sabemos informar endereços simples, para extrair a terceira linha do arquivo de usuários, basta fazer:

rac 3 /etc/passwd

É simples, basta informar diretamente o número da linha.
