RAC 5/12 - A Última Linha

Informando números negativos, a contagem é regressiva, feita a partir da última linha. 

  rac "-1" /etc/passwd

Este comando retornará a última linha do arquivo.

Da mesma forma, pode-se informar o endereço "-2 ; -3" para obter a penúltima e antepenúltima linhas.
