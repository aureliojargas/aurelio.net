MoneyLog Portable

Esta é a versão tudo-em-um do MoneyLog: em um único arquivo HTML você tem o app completo e os seus lançamentos de ganhos e gastos. Ela só funciona offline, e sua grande vantagem é a portabilidade.

Escolha esta se você geralmente está offline e precisa levar consigo suas finanças. Com o arquivo do MoneyLog na mão e um navegador, você está pronto.

Acesse a demonstração online: http://aurelio.net/moneylog/portable/demo/

Download

Salve em seu computador este arquivo HTML.

Dica: Se você usa Dropbox, salve o arquivo dentro dele, para poder acessá-lo de todos os seus computadores.

Agora abra este arquivo em seu navegador para começar a usar o MoneyLog Portable.

Nota para usuários do Internet Explorer: Se o MoneyLog não carregou corretamente e apareceu um aviso de segurança dizendo que ele pode ser perigoso ou algo assim, veja como proceder.

Atualizar MoneyLog da versão 4 para 5

Se você já é usuário do MoneyLog 4 e vai atualizar para o MoneyLog Portable 5, faça o seguinte:

  1. Baixe a versão 5.
  2. Abra os dois arquivos no editor de textos, o antigo (moneylog4.html) e o novo (moneylog5.html).
  3. Apague os lançamentos de exemplo que estão no final do arquivo novo.
  4. Copie seus lançamentos, que estão no final do arquivo antigo e cole lá no final do novo.
  5. Salve o arquivo novo.
  6. Pronto!

Inserir seus lançamentos

Seus dados ficam no mesmo arquivo HTML do MoneyLog Portable, você precisa editá-lo. É sempre lá no final do arquivo, não mexa em outras partes dele, ok?

Windows (Internet Explorer)

  1. Abra no Internet Explorer o MoneyLog Portable que você baixou.
  2. Acesse o menu Arquivo > Editar com Bloco de notas.
  3. Ele vai abrir o Bloco de Notas. Vá até o final do arquivo.
  4. Você vai ver os lançamentos de exemplo. Substitua-os pelos seus.

Linux

  1. Abra o arquivo HTML do MoneyLog Portable em um editor de textos simples, como Vim, Emacs, nano, gedit, Kate, ou qualquer outro editor de programação.
  2. Vá até o final do arquivo.
  3. Você vai ver os lançamentos de exemplo. Substitua-os pelos seus.

Mac

  1. Abra o arquivo HTML do MoneyLog Portable em um editor de textos simples, como Vim, TextMate, Smultron, TextWrangler, WriteRoom, BBEdit.
  2. Vá até o final do arquivo.
  3. Você vai ver os lançamentos de exemplo. Substitua-os pelos seus.
Atenção usuários do Editor de Texto (Aplicativos > Editor de Texto): Antes de abrir o MoneyLog nele, certifique-se de que a opção Ignorar comandos de texto com estilo em arquivos HTML esteja marcada.

O formato de cada lançamento é simples, usa apenas três campos: data, valor, descrição. Dúvidas? Consulte o Guia de Lançamentos.

Sempre que quiser conferir como está ficando, salve o arquivo no editor de textos e recarregue-o no navegador.

Você também pode importar dados de outros programas ou direto de seu Internet Banking. Utilize o conversor do MoneyLog, que entende os seguintes formatos: OFX, OFC, QIF (MS Money, Quicken). Saiba mais…

Pegadinha com <, > e &

Como você está editando um arquivo HTML, precisa respeitar algumas regrinhas que o formato impõe.

No caso, três caracteres são especiais e você deve evitá-los: <, > e &.

Se você realmente precisa colocá-los em seus lançamentos, terá que usar a versão "feia" deles:

  • &   →   &amp;
  • <   →   &lt;
  • >   →   &gt;
